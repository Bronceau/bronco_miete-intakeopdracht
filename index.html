Een stukje over mij.

Hallo ik ben Bronco Miete en ik heb de opleiding Software Developer gekozen omdat programmeren altijd al een hobby van mij is geweest.
En van je hobby je baan maken is iedereen zijn droom.
Daarom wil ik later graag als ICT Software Developer werken. Verder ben ik een persoon van oplossingen en niet van de problemen. Als ik een probleem heb probeer ik deze zo goed mogelijk bij te werken. In mijn vrije tijd zit ik veel achter mijn computer. Meestal ben ik aan het gamen.
Maar ik ben ook een Developer op een FiveM server van mijn vrienden.
FiveM is eigenlijk GTA V maar dan meer open source. Dit houd in dat je dingen kan toevoegen en ik doe dit veel in mijn vrije tijd.
De programmeer talen die je hiervoor nodig hebt zijn LUA, JS en natuurlijk een beetje HTML. Verder woon ik samen met mijn vader, moeder, broertje en 2 honden.
